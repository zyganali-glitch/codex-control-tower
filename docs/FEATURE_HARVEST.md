# Feature Harvest

Codex Control Tower uses the **Universal Agent OS family** as attributed concept research. “Harvested” means independently re-specified; it does not mean source code, UI, templates, screenshots, data, or assets were copied.

| Family concept | Codex Control Tower transformation | Status | Evidence |
| --- | --- | --- | --- |
| One-question alignment | English-only Phase-0 writes an alignment contract and first bounded mission. | YES | `cli/commands/phase0.js`, `core/en/`, Phase-0 tests |
| Plan before code | Repository findings become allowed scope, forbidden actions, required evidence, tests, and docs. | YES | Mission prompt builder and dashboard |
| Evidence-first closure | PASS/WARN/FAIL/NOT_RUN/SIMULATED stay distinct and named. | YES | Evidence collector, reports, tests |
| Brownfield caution | Existing repositories are scanned without overwriting unrelated work. | YES | Safe filesystem and repository scanner |
| Human review | Local Review Gate records explicit state and scope. | YES | Review-gate library, command, panel, tests |
| Explainable context | Context Trace shows selected items, reasons, freshness, importance, and protection. | YES | Context builder, command, panel, tests |
| Durable continuity | Flight Recorder and Memory Lens preserve events, rules, and minefields. | YES | JSONL recorder, memory analysis, dashboard |
| Mistake prevention | Proposed actions receive CLEAR/CAUTION/BLOCKED with reasons and a safer action. | YES | Mistake Shield and tests |
| Before/after proof | Prepared fictional InvoiceFlow Mini snapshots produce reproducible real scan/test results without claiming customer impact or a GPT-authored transformation. | YES | Demo fixtures, reports, dashboard |
| Real model collaboration | The featured explicit opt-in path invokes signed-in `gpt-5.6-sol` with medium reasoning read-only; it withholds locked comparison targets/expected classes, collects counter/missing evidence, and escalates policy conflict without changing facts. | YES | `codex-review` command, audit panel, contract tests, recorded evidence files |
| Multilingual packs | Additional languages beyond the English Build Week version. | FUTURE | Roadmap only |

## Explicitly not harvested

- No family implementation, template, frontend, screenshot, test data, cloud endpoint, credential, or branding.
- No fake embeddings, remote graph results, enterprise authorization, deployment proof, or award claim.
- No family repository is a runtime dependency or demo target.
