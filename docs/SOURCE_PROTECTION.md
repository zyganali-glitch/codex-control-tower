# Source Protection and Lineage Record

Codex Control Tower is an independent OpenAI Build Week project. It acknowledges concept-level inspiration from the **Universal Agent OS family**. The family was used only as protected, read-only research; it was never the working repository, package identity, runtime dependency, or demo target.

The primary family source is [Universal Agent OS](https://github.com/zyganali-glitch/Universal-Agent-OS). Public pages and selected public files were reviewed read-only on 2026-07-12. No family repository was cloned into this workspace, modified, renamed, deleted, or pushed to.

## Boundary record

| Category | Record |

## Protection lock

- Read family sources only; never write to a family remote or checkout.
- Never use a family source as a scan demo, generated-work target, or deployment target.
- Keep package identity, commands, UI, reports, and screenshots specific to Codex Control Tower.
- Use only InvoiceFlow Mini or another explicitly approved fictional target for demonstrations.
- Do not import family screenshots, branding, hosted endpoints, credentials, private URLs, or evidence.
- If future work copies a substantial fragment, record the exact source, license, attribution, transformation, and destination before release. The current record is: **no such copying**.
