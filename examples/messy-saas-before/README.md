# InvoiceFlow Mini

> SIMULATED DEMO DATA — this is not a real customer project.

Tiny invoice tracker. Start it with `npm start`.

The app keeps customers, invoices, and payment records in memory. Authentication
is temporary and payments are wired for the demo. Production launch was planned
for spring 2024.

Mostly done. Tests still need attention.
